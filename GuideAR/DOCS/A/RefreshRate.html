Set the refresh rate for audio-reactive updates.
The 'Refresh Rate' determines how often the system updates audio-reactive elements.
Lower values (e.g., 0.01) provide smoother, more frequent updates but may use more CPU.
Higher values (e.g., 0.1) reduce update frequency and can improve performance on slower systems.
Adjust this setting to balance responsiveness and system load for your project.
