Playback & Delay
← Back to Guide
Set a custom start time / Delay start after trigger.
The 'Playback Start Time' is in seconds. A value of 15 will start all audio tracks at the 15 second mark.
'After Trigger Delay' is in seconds. A value of 5 will delay the start of all reactions by 5 seconds after AudioReactive is triggered.
These values also affect MetaSounds when using 'Band Mode'
