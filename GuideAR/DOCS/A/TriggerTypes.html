AudioReactive Uses BeginPlay by default to trigger.
Sequencer Trigger will allow you to use an Event Trigger in sequencer to trigger AudioReactive.
The Collision Trigger by default will trigger when your player collides with the AudioReactive actor in the level.
You can easily customize the collision trigger and attach it to projectiles – This method works great with the Rapid Fire feature.
Both Sequencer and Collision triggers will disable the BeginPlay trigger.
