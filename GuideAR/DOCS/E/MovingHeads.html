Moving Head Intro
Moving Head fixtures have extra functions. They have the standard Dimmer/Shutter/Zoom/Frost controls, as well as the Pan and Tilt functions unique to this fixture. Each function has it's own Audio Track Override as well as a "Mute Function" checkbox for detailed per function control.
Using Pan and Tilt with audio reaction can be very successful,
if you set your Audio Smoothing settings correctly. Audio that has not been properly smoothed will make the lights move with extremely high acceleration.
Moving Heads also use the ColorWheel, instead of RGB values
as found in the Strobe, Wash, "Laser", and Static fixtures
Using a colorwheel means you are setting a value based on a pre-determined chart that assigns value ranges to specific colors, colorblends, and colorwheel animation presets.
They also use a gobo wheel, with similar setup as the colorwheel, but instead of colors, you are selecting a gobo pattern, with certain ranges being animated gobo presets.
These fixtures utilize the default DMX wheels that come with the engine, and can be found in either the plugins folder, or I've made duplicates inside the AudioReactive/DMX folder.
Moving Head Midtro
Moving Heads use a few unique parameters that have been added a sub-rollout,
"Moving Head Attributes" inside any DMX Fixture index.
Control the ColorWheel, Gobo, and Tilt Offset values. Tilt Offset will add a positive or negative value (30/-30) and use that as the starting point for audioreaction.
If Audio Reaction is disabled on your Tilt Function, the Tilt Offset will instead be used as a static value to control the tilt.
You can create your own custom ColorWheel / Gobo datatables and take full control over the values and animations.
Any alpha style image texture can be used as a gobo, and any color can be used in the colorwheel.
Duplicate the existing datatables, adjust to your liking, and update the BP_MovingHead_AR's ColorWheel and Gobo Components to point to your new datatables.
Moving Head Outro
There are a few shared attributes that affect multiple fixtures types.
The parameters 'Shutter' and 'Enable Beam' are used across all fixtures.
Shutter controls the strobe affect, and like color and gobo, has it's own value chart full of preset animated effects.
Enable beam, well, it enables the beam. The awesome volumetric beam that reacts to the "Zoom" function of your fixtures can be enabled/disabled with a single click.
If you want to adjust the quality of the beam, for now you need to open the Blueprint 'BP_MovingHead_AR', select BP_MovingHead_AR root, and in details, you'll see the low/medium/high/ultra/custom presets.
This option is available per fixture in level, but if you want them all to update, adjust in the blueprint and compile.
