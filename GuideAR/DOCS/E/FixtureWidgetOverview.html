Fixture Widget Setup:
The DMX Fixture Widget allows you to quickly group and arrange fixtures and assign them AudioReactive parameters. You can 'Select All Fixtures Of Type' to quickly select all fixtures of a certain type.
..but first, those annoying fixture beam cone visualizers make it very difficult to work - simply press ALT+R to enable/disable them.
You can use the 'Add Actor Tag On Selected Fixtures' to add the tag. Duplicate tags are currently NOT checked (in progress).
Once your Fixture Type dropdown is set, you can now see the '2. Select Actor Tag' is populated with all found Actor Tags in your level for that Fixture Type.
Once the tag dropdown is selected, you can now select fixtures of that type, by their actor tag.
If you want to clear all and start fresh, you can utilize the "Clear ALL ACTOR TAGS On Selected Fixtures" button to delete all actor tags on selected fixtures.
Actor Tags can also be added manually via the details panel for any fixture you have selected. Simply click the + button under the Actor Tag rollout in the Details of the fixture actor, and add the tag. Once added manually, the DMX Widget will automatically update the dropdown to include any manually added tags.
Quickly Update Standard Fixtures To AudioReactive:
One of my favorite buttons, "Update All Fixtures To AudioReactive", will quickly swap out the standard engine version of the following fixtures to the AudioReactive versions:
Moving Head - Static Head - "Laser" - WashLED - Strobe
All existing fixture positions will be preserved, so you won't lose your layout.
